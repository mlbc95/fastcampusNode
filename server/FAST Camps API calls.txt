/////////////////////////////////////////////////////AUTH API/////////////////////////////////////////////////////////////////////////////////

URL: auth/profile
Request: GET
Header type: Authorization + token 
Returns: user info by passing token in the header (look at setting page)



/////////////////////////////////////////////////////USERS API////////////////////////////////////////////////////////////////////////////////

URL: users/updateUser
Request: POST 
Body: {
	"fName":"Mohamed",
	"lName": "Langi",
	"school": "UMSL",
	"id": "5974ce963b3a1b17e09ab8e4"
}
Description: Updates user first/last name and school. Must pass the ID in the body

URL: users/getAllUsers
Request: GET
Returns: All user info except password and _ID
